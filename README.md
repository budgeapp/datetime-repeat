# datetime-repeat

[![PyPI - Version][badge/pypi]][pypi]
[![Test with pytest][badge/pytest]][workfows/pytest]
[![Publish release to PyPI][badge/pypi-publish]][workflows/pypi-publish]

A library to create and iterate through complex sequences of repeating dates and
times in Python.

## Installation

```bash
pip install datetime-repeat
```
